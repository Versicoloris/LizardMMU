🦎 LizardMMU (Mount Me Up)

Tiny QoL addon for World of Warcraft.

Pick your favorite mounts and summon one at random with a single click.

**✨ What It Does**
🎲 Random mount from your favorites
⭐ Custom mount list (you choose)
🔁 Automatically skips unusable mounts
🖱️ Clickable UI button (movable)
🪶 Lightweight


🕹️ Usage

Add mounts (exact name):
/lmmu add Headless Horseman's Mount

Remove:
/lmmu remove Headless Horseman's Mount

List:
/lmmu list

Clear all:
/lmmu clear

Summon:
/lmmu

🖱️ Button Controls
Left Click → Summon random mount
Right Click → List favorites
Shift + Click → Toggle current mount
Drag → Move button

📦 Installation
Download the latest release
Extract into:
Interface/AddOns/
Restart the game
